# claude-ios-arch

A Claude/agent-oriented iOS architecture package for greenfield and early-stage apps.

## What it includes
- `CLAUDE.md` top-level architecture contract
- `rules/` for low-debt and architecture guardrails
- `commands/` for repeatable architecture/review workflows
- `skills/ios-app-bootstrap/` for reusable agent guidance
- `hooks/ios_arch_guard.py` for lightweight heuristic checks
- `plugins/ios-app-bootstrap/` plugin scaffold manifest

## Focus
- SwiftUI-first baseline
- async/await concurrency
- feature-first structure
- clean-ish layering
- low-debt startup guidance
- staff-level architecture reviews

## Current status
This repository is a v1 scaffold intended to evolve into a stronger agent/plugin package.
