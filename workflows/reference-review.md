# Reference Review Workflow

## Goal
Update a single reference while preserving clear provenance.

## Steps
1. Open the official sources named in the registry.
2. Prefer primary documentation, official repo READMEs, official migration guides, and official release notes.
3. If one official source is inaccessible because of network/policy limits, use another official source for the same product instead of falling back to random blogs.
4. Check whether the current reference still matches the available official material.
5. Update recommendation, trade-offs, and caveats if needed.
6. Update `Last reviewed` and confidence.
7. Record related skills that depend on this reference.
