# /ios-discovery

## Intent
Use when the user wants help with the **需求 / discovery** phase of an iOS app.

## Default Mode
Manual-trigger, user-in-the-loop.

## Typical Tasks
- problem framing
- target user clarification
- MVP scope definition
- user flow definition
- feasibility/risk framing
- App Store/policy sensitivity review

## Output Format
- problem statement
- target user
- must-have use cases
- MVP boundary
- top risks
- recommended next step
