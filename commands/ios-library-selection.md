# /ios-library-selection

## Intent
Use when deciding whether to keep the official baseline or introduce a third-party library.

## Agent Workflow
1. Determine whether the official platform capability is sufficient.
2. Evaluate real project constraints and complexity.
3. Score the library choice against maintenance, migration, testing, and lock-in costs.
4. Recommend keep official / adopt library / retain existing library.
5. State future replacement triggers.

## Output Format
- Problem summary
- Official baseline evaluation
- Options considered
- Recommendation
- Trade-offs
- Replacement / exit trigger
