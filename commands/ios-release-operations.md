# /ios-release-operations

## Intent
Use when the user wants help with the **发布运营 / release / post-launch** phase of an iOS app.

## Default Mode
Manual-trigger, user-in-the-loop.

## Typical Tasks
- TestFlight readiness
- App Store Connect readiness
- release checklist
- submission preparation
- launch monitoring checklist
- post-launch issue triage

## Output Format
- current release stage
- required actions
- policy/compliance/watchouts
- launch checklist
- post-launch monitoring focus
