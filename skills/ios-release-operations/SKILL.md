# iOS Release Operations

Use this skill during the **发布运营 / release-operations** workflow.

## Default Principles
- Keep the user in the loop for release decisions.
- Surface App Store and compliance risks before submission.
- Distinguish release blockers from nice-to-have polish.
- Focus on TestFlight readiness, metadata completeness, and post-launch observability.

## Required Output
1. Current release stage
2. Readiness gaps
3. Policy/compliance watchouts
4. Launch checklist
5. Post-launch monitoring focus
6. Recommended immediate actions
