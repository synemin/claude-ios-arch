# iOS Platform Guide

This plugin currently supports **iOS only**.

## Platform Scope
Use this plugin for:
- SwiftUI/UIKit architecture design
- iOS app technical selection
- directory structure and feature decomposition
- low-debt startup guidance
- architecture review and debt checks

## Default Baseline
- SwiftUI first
- async/await first
- feature-first structure
- clean-ish layering
- initializer injection
- URLSession typed client
- explicit model boundaries
- day-1 observability

## Not Yet Covered
- Android-specific guidance
- Flutter-specific guidance
- React Native-specific guidance
- multi-platform abstractions beyond high-level roadmap comments
