# iOS Release Checklist

## Beta Readiness
- core happy path works
- crash reporting enabled
- analytics/events visible
- major blocker issues triaged
- TestFlight notes prepared if needed

## App Store Connect Readiness
- app metadata complete
- screenshots ready
- privacy policy URL ready
- app privacy / data collection filled
- categories / age rating / pricing set
- in-app purchase or subscription config checked if relevant

## Submission Readiness
- permissions strings match app behavior
- account flows comply with Apple requirements
- review notes prepared for any non-obvious behavior
- release build verified from clean archive/upload path

## Post-Launch Readiness
- crash monitoring owner exists
- launch metrics defined
- support / feedback path exists
- rollback / hotfix plan exists
